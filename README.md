# SEEDAS-ピッチ用ランディングページ
1/9に行われるKwansei Gakuin Pitch Contestで使うためのLPを作成

## キーワード

- レスポンシブデザイン
- Vue.js / Nuxt.js / SCSS

## Build Setup

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start

# deploy to GitHub Pages
$ yarn deploy
```
