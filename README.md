# ランディングページ サンプル

これはポートフォリオ用LPのソースです。

完成形はこちらでご覧ください → https://pinalto.github.io/demo-lp/ 

## キーワード

- レスポンシブデザイン
- Vue.js / Nuxt.js / SCSS

## Build Setup

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start

# deploy to GitHub Pages
$ yarn deploy
```
